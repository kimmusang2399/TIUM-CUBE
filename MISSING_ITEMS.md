# 추가로 정하면 좋은 항목

현재 기획서만으로도 랜딩페이지 개발은 시작할 수 있지만, 완성도를 높이려면 아래 항목을 추가로 정하는 것이 좋습니다.

## 필수에 가까운 항목

- 실제 연락처
- 카카오톡 상담 링크
- 회사 로고
- 실제 제품 이미지 또는 대표 이미지
- 회사 기본 정보
- 개인정보 처리방침 필요 여부

## 마케팅 관련

- 검색 노출용 페이지 제목
- 검색 설명 문구
- 공유용 오픈그래프 이미지
- 대표 키워드
- 상담 전환 목표
- 광고 유입 여부

## 신뢰 확보 자료

- 실제 설치 사례
- 예상 비용 범위
- 유지보수 방식
- 기술 인증 또는 특허 여부
- 파트너사 또는 협력 기관
- 고객 후기

## 운영 관련

- 상담 접수 후 응대 방식
- 전화 상담 가능 시간
- 카카오톡 응답 가능 시간
- 문의 수집 방식
- 배포 도메인
- 유지보수 담당자
