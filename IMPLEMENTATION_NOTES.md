# TIUM CUBE 구현 메모

## 구현 범위

- HTML
- CSS
- JavaScript
- 반응형 랜딩페이지
- 스크롤 애니메이션
- Hover 효과
- 전화 및 카카오톡 링크

## 제외 기능

- 로그인
- 회원가입
- 데이터베이스
- 관리자 페이지
- API
- 결제
- 게시판

## 권장 파일 구조

```text
/
├─ index.html
├─ assets/
│  ├─ images/
│  └─ icons/
├─ css/
│  └─ style.css
├─ js/
│  └─ main.js
└─ docs/
```

## 권장 섹션 순서

1. Header
2. Hero
3. 왜 큐브 스마트팜인가
4. 기존 스마트팜 VS 큐브 스마트팜
5. 핵심기술
6. 도입효과
7. 적용분야
8. 사업 추진 절차
9. 상담문의
10. Footer

## 추후 확정이 필요한 정보

- 실제 전화번호
- 실제 카카오톡 상담 링크
- 회사명 표기 방식
- 주소
- 사업자등록번호
- 대표자명
- 이메일
- 실제 제품 이미지 또는 렌더링 이미지
- 로고 파일
- 도메인
- 배포 방식
